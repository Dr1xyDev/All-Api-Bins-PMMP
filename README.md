# 🐧 Linux Bins PocketMine PM2 - PM3 - PM4

- 📦 este repositorio contiene binarios de PHP para Linux para usar con PocketMine-MP y forks basados en PocketMine.

- 🧰 Aquí se guardan varias versiones de PHP ya compiladas para que sea más fácil ejecutar servidores que requieren una versión específica de PHP.

---

📚 Información

Con el tiempo, muchas versiones de PocketMine y sus forks han usado versiones diferentes de PHP.
Por eso a veces es difícil ejecutar ciertos núcleos si no se tiene el PHP correcto.

Este repositorio guarda binarios de diferentes versiones de PHP para que sea más fácil usar esas versiones de servidores.

---

⚠️ Importante

Estos binarios solo funcionan con PocketMine-MP o forks que requieren específicamente esas versiones de PHP.

Si tu servidor usa otra versión de PHP, debes usar el binario que corresponda a esa versión.

---

Los binarios fueron compilados solo para:

- 🐧 Linux

---

🚀 Uso

1. 📥 Descarga la versión de PHP que necesite tu servidor.
2. 📂 Coloca el binario en la carpeta de tu servidor y no sufrirás de errores de ```permission-denied``` por que ya esta se puede usar en cualquier host de panel pterodactyl y en VPS y VDS.
3. ▶️ Ejecuta el servidor normalmente con el "start.sh" de tu PocketMine o si estas en panel ptero solo sube el binario y tu PocketMine-MP.phar y descomprime el binario.
